# Kernel-Basico
Kernel básico multitarea, diseñado para correr en qemu emulando una IBM-PC. Desarrollado durante la cursada de la materia Arquitectura y Organización del Ordenador de la FCEN UBA

#### Software necesario para ejecutar este kernel en una máquina virtual:
- GCC
- GDB
- NASM
- QEMU
- Valgrind

Para ejecutarlo simplemente correr en una terminal en la carpeta ´src´ el comando ´make qemu´
